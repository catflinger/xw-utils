Browser compatability

So far I have run the application successfully on Windows with Chrome, Firefox, Opera and Edge browsers.I have also used the application successfully on a mobile phone running Android with Chrome.

The application will function on Internet Explorer 11 on Windows but has display issues where the text and images overlap on some screen sizes.

I have not tried with Safari or with any browser runnning on an Apple device. Please let me know if you have an Apple device and it works for you. I have not tried the application in any browser under Linux yet.
